CSS Display Modes: Block, Inline, and Inline-Block

This page visually illustrates the rendering differences between the three main display properties.

1. Display: Block (display: block)

Block elements stack vertically, take up 100% of the parent container's width by default, and fully respect set width, height, margins, and padding.

Block Box 1
Block Box 2
  • Line Break: Starts on a new line.
  • Width/Height: Fully respected (set to 150px × 60px).
  • Margins: Vertical margins (set to 15px) create space above/below.

2. Display: Inline (display: inline)

Inline elements sit side-by-side on the same line, wrapping like text. They ignore width and height properties. Vertical margins and padding do not push other elements away.

Inline Box 1 Inline Box 2 Inline Box 3
  • Line Break: Does NOT start on a new line (sits side-by-side).
  • Width/Height: Ignored (the boxes only wrap their text despite being set to 150px × 60px in CSS).
  • Margins: Vertical margins are ignored; horizontal margins are respected.

3. Display: Inline-Block (display: inline-block)

Inline-block elements flow side-by-side on the same line like inline elements, but they fully respect width, height, margins, and padding like block elements.

  • Margins: Vertical and horizontal margins are fully respected (set to 10px).

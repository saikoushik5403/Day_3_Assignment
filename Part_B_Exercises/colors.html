CSS Color Formats Demonstration

This page displays the exact same color base (Red / Coral) rendered using five different CSS color specifications.

1. Named Color

Value: color: red; or background-color: red;

Uses one of the 140 English color names built directly into the browser standard.

2. HEX Format

Value: background-color: #ff0000;

Hexadecimal representation of Red, Green, and Blue intensities from 00 to ff.

3. RGB Format

Value: background-color: rgb(255, 0, 0);

Specifies color intensities numerically for Red, Green, and Blue channels from 0 to 255.

4. HSL Format

Value: background-color: hsl(0, 100%, 50%);

Defines color using Hue (angle on color wheel 0-360), Saturation (percentage), and Lightness (percentage).

Overlay Text

5. RGBA Format (Transparent)

Value: background-color: rgba(255, 0, 0, 0.4);

Adds a fourth value (Alpha channel) from 0.0 (fully transparent) to 1.0 (fully opaque) to create transparency overlays.

CSS Selectors Demonstration

This page demonstrates all nine major CSS selectors with comments in the stylesheet explaining each.

1. Universal Selector (*)

Every element on this page is styled with box-sizing: border-box and clean margins using the universal selector *.

2. Element Selector (p)

This paragraph is styled using the element selector targeting all <p> tags. It has a customized font line-height and color.

3. Class Selector (.card)

This is a class-styled card

Styled with the class selector .card, applying a border, padding, and subtle shadow.

4. ID Selector (#main-header)

The top page header has a dark slate background, centered text, and custom padding because it is styled using the ID selector #main-header.

5. Group Selector (h1, h2)

All <h1> and <h2> headings share the same font family and blue color because they are grouped in the stylesheet.

6. Descendant Selector (div p)

This paragraph is inside a .descendant-demo div. The descendant selector (space) targets any paragraph nested inside this div.

This span is not targeted.

7. Child Selector (.parent > p)

This paragraph is a direct child of .parent-demo and turns green.

This paragraph is nested inside a sub-div (grandchild), so it does NOT match the child selector and remains standard.

8. Adjacent Sibling Selector (h2 + p)

An h3 Heading (Sibling)

This paragraph immediately follows the <h3> heading, so it is styled with a bold weight using the adjacent sibling selector +.

This paragraph is a sibling but does not immediately follow the heading, so it is normal weight.

9. General Sibling Selector (h4 ~ p)

An h4 Heading (Sibling Anchor)

This sibling paragraph gets styled with purple text.

An unrelated sibling span.

This sibling paragraph also gets styled with purple text because it follows the h4 sibling anchor (even though a span is in between).

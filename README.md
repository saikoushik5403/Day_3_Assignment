# CSS3 Fundamentals - Day 3 Assignment

This repository contains the complete solutions for the Day 3 CSS3 Fundamentals assignment as outlined in the course material (Section 28, Page 105).

## Project Structure

The project is organized into the following directories:

*   **`Part_A_Theory/`**: Contains answers to the 10 theory questions about CSS, selectors, display types, box models, positioning, specificity, and responsiveness.
*   **`Part_B_Exercises/`**: Contains 5 practical exercises demonstrating:
    *   `selectors.html` & `selectors.css`: Demonstrating the 9 selector types.
    *   `box_model.html` & `box_model.css`: Demonstrating the box model and `box-sizing: border-box`.
    *   `display_modes.html` & `display_modes.css`: Demonstrating `block`, `inline`, and `inline-block`.
    *   `positioning.html` & `positioning.css`: Demonstrating all 5 position types (including sticky headers and absolute badges).
    *   `colors.html` & `colors.css`: Demonstrating color formats (HEX, RGB, HSL, RGBA transparency).
*   **`Part_C_Mini_Project/`**: A styled, fully responsive Personal Portfolio page representing the final practical assignment. It features a sticky navigation menu, card layouts, a circular profile photo, interactive hover transitions, and mobile media queries.
*   **`Part_D_Research/`**: Contains findings from research on CSS variables, BEM naming conventions, advanced pseudo-classes, and CSS Flexbox layout basics.
